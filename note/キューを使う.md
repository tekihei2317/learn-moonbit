# キューを使う

標準ライブラリに@queueがあります。

```mbt
///|
test "queue example" {
  let queue = @queue.new()
  for i in 0..<5 {
    queue.push(i)
  }
  let arr = []
  while !queue.is_empty() {
    let value = queue.unsafe_pop()
    arr.push(value)
  }
  inspect(arr, content="[0, 1, 2, 3, 4]")
}
```

こっちの方が綺麗。`is Some(var)`で、Someかどうか判定しながら中身を取り出せる。

```mbt
///|
test "queue example 2" {
  let queue = @queue.new()
  for i in 0..<5 {
    queue.push(i)
  }
  let arr = []
  while queue.pop() is Some(value) {
    arr.push(value)
  }
  inspect(arr, content="[0, 1, 2, 3, 4]")
}
```
