# ライブラリを作るときのディレクトリ構成

[自分が Moonbit 言語について知っていること](https://zenn.dev/mizchi/articles/practical-moonbit#%E3%83%A9%E3%82%A4%E3%83%96%E3%83%A9%E3%83%AA%E3%81%AE%E4%BD%9C%E3%82%8A%E6%96%B9)

`moon.pkg.json`と同じ階層にある`*.mbt`ファイルが、そのパッケージに含まれる。

なので、そこで`pub`をつけた、パッケージとして公開される。

```text
$ tree ~/repo/peter-jerry-ye/memory/
├── LICENSE
├── README.md
├── memory.mbt
├── moon.mod.json
└── moon.pkg.json
```

上記のように、トップレベルにmoon.pkg.jsonとmoon.mod.jsonをおくのが一番シンプル。
