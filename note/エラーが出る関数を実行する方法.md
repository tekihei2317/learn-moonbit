# エラーが出る関数を実行する方法

Stringのsliceが範囲外のときに、`Function with error can only be used inside a function with error types in its signature. Please fix the return type of this function.`エラーが出る。

エラーハンドリングの方法を知る必要がある。

シグネチャに`raise Error`または`raise`をつければ、エラーが出る処理をそこで書ける。

ただ、どこかでハンドリングしないといけなくて、その場合の一般論がいまいち分からない...。

`guard`ってなんだろう。
